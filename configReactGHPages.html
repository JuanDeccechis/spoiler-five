React app

For create a react app you need node and npm

Then write in a console, npm i create-react-app, and finally npx create-react-app my-app

An alternative is create-react-redux-router-app, that is more specific.

Write in the console cd my-app and npm start to run it



React app in gh-pages

https://dev.to/yuribenjamin/how-to-deploy-react-app-in-github-pages-2a1f

In the console, write npm install gh-pages --save-dev

Then go to my-app -> package.json. And edit it with:

"homepage": "http://juandeccechis@github.io/myapp", and in the scripts section: "predeploy": "npm run build", "deploy": "gh-pages -d build",

And finally write in the console, npm run deploy, for launch it.
